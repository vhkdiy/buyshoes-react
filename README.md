# 项目介绍
> 使用React技术栈开发SPA购物车

# 使用
1.下载 `buyshoes` project:

```
git clone https://github.com/vhkdiy/buyshoes-react.git buyshoes
```

2.安装

```
npm install
```

3.开启server

```
make server
```

# 技术栈
 - UI: Bootstrap
 - 框架: React & Redux & Immutable
 - JS: ES6
 - 打包构建: Babel Webpack
 - 包管理：Npm
